## Prerequisites

- npm

```sh
sudo npm install npm@latest -g
```

- node-red

```sh
sudo npm install -g --unsafe-perm node-red
```

## Installation

Run the following command on a command-line interface.

```sh
npm install @hypertegrity/node-red-contrib-k8s-helper
```
